# Homepage-Back

### How to contribute?

- KEEPER 홈페이지의 발전에 기여해주세요!
- KEEPER 홈페이지 백엔드 프로젝트는 2022학년도 1학기 이후, 오픈소스로 운영됩니다.

- 아래 3개의 문서를 읽고 Pull request를 하실 때 기능별로 지정된 Reviewer를 등록해주시면 됩니다!

---
